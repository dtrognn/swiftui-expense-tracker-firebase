# SpendSmart

> An expense management application, created with the aim of improving skills.

---

### Some features

- Authentication: Register, Login, Logout
- CRUD transactions
- CRUD categories
- CRUD savings
- Analyze transactions using graphs
- Change languages
- Upload image
- Dark mode (updating)
